Mobile 2D-Platformer

Presents mobile features of DPad (only left and right UI buttons), single tap function (small jump), press hold function (for charge jump), and swipe function (to perform a mid-air dash).
Enemies are present in the form of stationary drones that fire projectiles which can damage the player.
At the moment there is only one level present.

By: Diego Morales
